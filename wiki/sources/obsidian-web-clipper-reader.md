# Obsidian Web Clipper Reader

## 摘要

这个来源说明 Obsidian Web Clipper 的 Reader 模式。Reader 会把网页替换成更干净
的阅读视图，保留主要文章内容、图片和格式，并提供大纲、代码高亮、脚注弹出层
等阅读辅助能力。

原文路径：[raw/inbox/Reader.md](../../raw/inbox/Reader.md)

## 关键观点

- Reader 是 Web Clipper 内置的阅读模式，目标是去除网页杂讯，突出正文。
- 用户可以通过扩展面板的书本图标、快捷键或网页右键菜单启用，也可以把扩展按钮
  设置成一键打开 Reader。
- Reader 抽取的视图包含标题、作者、发布日期、来源域名、正文内容、图片和格式；
  多标题文章会显示侧边栏大纲。
- Reader 支持代码块语法高亮和可点击的脚注弹出层。
- 阅读视图可配置字体、字号、行高、行宽、明暗外观、主题和自定义 CSS。

## 有用细节

- 这个来源对 source capture 工作流有直接价值：它说明 Web Clipper 在保存前可以
  先把网页转成更稳定的阅读视图。
- Reader 的入口和退出都在同一组操作路径中，来源强调这些操作可以恢复原网页。

## 张力

- 这是功能帮助页，不是抽取准确率或边界条件评测；不能据此断言 Reader 对所有网页
  都能正确识别正文。
- 该来源只覆盖 Reader 模式，不覆盖 Web Clipper 的模板、保存目标或 metadata
  映射规则。
