# Obsidian Web Clipper

## 摘要

Obsidian Web Clipper 是 Obsidian 生态中的网页剪藏工具。当前 wiki 只摄取了它的
Reader 模式帮助页，因此这里先聚焦于它对网页阅读和来源整理的意义。

## 已知事实

- Web Clipper 包含一个名为 Reader 的内置阅读模式，用于去除网页杂讯并展示主要
  文章内容。证据：[[sources/obsidian-web-clipper-reader|Obsidian Web Clipper Reader]]
- Reader 可以从扩展面板、快捷键、网页右键菜单或扩展按钮的一键打开设置进入。
  证据：[[sources/obsidian-web-clipper-reader|Obsidian Web Clipper Reader]]
- Reader 视图会保留正文图片和格式，并支持大纲、代码高亮、脚注弹出层和阅读外观
  自定义。证据：[[sources/obsidian-web-clipper-reader|Obsidian Web Clipper Reader]]
