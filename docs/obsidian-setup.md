# Obsidian 设置

把仓库根目录作为 Obsidian vault 打开。

推荐设置：

- Files and links：把附件文件夹路径设为 `raw/assets/`。
- Files and links：启用文件重命名时自动更新链接。
- 核心插件：启用 Graph view、Backlinks、Search 和 Templates。
- Templates：把模板文件夹位置设为 `templates/`。

有用的社区插件：

- Dataview：跨 wiki 页面查询 YAML frontmatter。
- Marp Slides：需要时从 Markdown 生成幻灯片。
- Web Clipper：把文章保存到 `raw/inbox/`。

把来源材料放在 `raw/`，把生成页面放在 `wiki/`。这样既能保持图谱清晰，又能
保留来源脉络。
