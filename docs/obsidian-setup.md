# Obsidian Setup

Open the repository root as an Obsidian vault.

Recommended settings:

- Files and links: set attachment folder path to `raw/assets/`.
- Files and links: enable automatic link updates when files are renamed.
- Core plugins: enable Graph view, Backlinks, Search, and Templates.
- Templates: set template folder location to `templates/`.

Useful community plugins:

- Dataview: query YAML frontmatter across wiki pages.
- Marp Slides: generate slide decks from Markdown when needed.
- Web Clipper: save articles into `raw/inbox/`.

Keep source material in `raw/` and generated pages in `wiki/`. This keeps the
graph readable while preserving provenance.
